To execute this project, follow the below steps:
1. Clone the project to your local machine
2. Copy CCAutomation to your drive.
3. In TestSuite.java file update ChromeDriver path
4. In ExtentFactory.java file update report path
5. To run the project:
	1. right click on testng file and run as testng project
	2. right click on project and run as maven test (before this, you might want to run maven clean, maven build)
